# S.H.I.T.S — 部署指南

## 本地预览（直接打开）
双击 `index.html`，用浏览器打开即可。四个页面之间的跳转完全正常。

---

## 部署到 GitHub Pages（免费公网访问）

### 第一步：开通 Formspree（投稿表单）
1. 前往 https://formspree.io 注册免费账号
2. 点击 **New Form**，名字随便填
3. 复制表单 ID（格式：`xpwzabcd`）
4. 打开 `submit.html`，找到这一行：
   ```
   action="https://formspree.io/f/YOUR_FORM_ID"
   ```
   把 `YOUR_FORM_ID` 替换为你的真实 ID

### 第二步：上传到 GitHub
1. 前往 https://github.com，登录或注册
2. 点击右上角 **+** → **New repository**
3. 仓库名：`shits-journal`（或任意名称）
4. 选 **Public**（必须公开才能用免费 Pages）
5. 点击 **Create repository**
6. 按页面提示，把 ZHENGHUO 文件夹里的所有文件上传到仓库根目录

   方法一（网页上传）：
   - 点击 **uploading an existing file**
   - 把 index.html / papers.html / submit.html / about.html / css/ / js/ 全部拖进去
   - 点击 **Commit changes**

   方法二（命令行，如果装了 Git）：
   ```bash
   cd 你的ZHENGHUO目录
   git init
   git add .
   git commit -m "initial commit"
   git remote add origin https://github.com/你的用户名/shits-journal.git
   git push -u origin main
   ```

### 第三步：开启 GitHub Pages
1. 进入仓库页面，点击顶部 **Settings**
2. 左侧找到 **Pages**
3. Source 选择 **Deploy from a branch**
4. Branch 选 **main**，目录选 **/ (root)**
5. 点击 **Save**
6. 等待约 1-3 分钟，页面会显示你的网址：
   ```
   https://你的用户名.github.io/shits-journal/
   ```

## 部署到 Vercel（更快，推荐）

1. 前往 https://vercel.com，用 GitHub 账号登录
2. 点击 **Add New Project**
3. 选择你刚才创建的 `shits-journal` 仓库
4. 所有设置保持默认，点击 **Deploy**
5. 完成后会给你一个 `.vercel.app` 域名，立即可访问

---

## 更新内容

日后想修改论文、改文案，直接编辑对应 HTML 文件，重新上传覆盖即可。
如果用命令行：`git add . && git commit -m "update" && git push`，网站会自动更新。
